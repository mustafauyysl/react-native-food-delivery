# hungryBear
Food Delivery App - React Native, Redux, Redux Thunk, JSON Server

# Installation

Firstly, run JSON Server

```
   npm intall json-server
   
   cd hungryBear && cd api
   
   json-server --watch db.json
   
```

Now, run project
